# App web: Gestión de Accesorios FMC

> Es una aplicación web simple y ligera diseñada para digitalizar y optimizar el control de las paletas de accesorios para las unidades en producción. El sistema sirve como plataforma centralizada que garantiza la trazabilidad de los componentes, desde su salida del área hasta su entrega final en el proceso de acabados.

## Características

- Validaciones de Seguridad: Sistema robusto para evitar duplicidad de guías o registros incompletos.
- Emisión de Guías Digitales: Registro rápido de movimiento de accesorios con generación de ID automático.
- Historial de Movimientos: Auditoría completa de cuándo y cuales accesorios fueron movilizados.
